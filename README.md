## **`Linked-In-Clone`**

### Summary

This projects deals with re-building the linked-in application from scratch using `REACT`

### Technical stack

- Node
- React + React Hooks
- Redux
- Material UI
- React Spring
- Lottie Animations
- Flexbox
- Firebase
  - Firestore real time DB
  - Firebase Hosting
  - Firebase Google Authentication
